# Question
## Remove space
### Problem

Remove space each line.

### Sample input:_

```console
c a r b o n
```

### Sample output:

```console
carbon
```

### Sample input:_

```console
carbo n
nitroge n
```

### Sample output:

```console
cargon
nitrogen
```
